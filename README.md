# Проект "Вычислитель отличий":
Gendif- это приложение которое выводит разницу между структурой двух файлов. 
Поддерживаются файлы форматов JSON, YAML.
# Установка 
```
git clone git@github.com:Moonlin14/frontend-project-46.git
```
```
make install
```
```
npm link
```
